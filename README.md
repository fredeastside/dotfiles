## Tmux
C+b+c - create a new window

C+b+& - close a window

C+b+n - switch current window

C+b+NUMBER - switch to NUMBER window

C+b+% - split vertically

C+b+” - split horizontally

C+b+ARROW - move between panes

C+b+: - go to command mode

:rename-window - rename a window

C+b+d - detach

tmux ls - show session list

tmux attach - restore session

C+b+s - show open session

:rename-session - rename a window
